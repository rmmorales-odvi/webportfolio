Inspired by the bootstrap grid system, Mors wanted to design a project portfolio in the style of a magazine layout. This means big images, bold fonts and multiple sections and columnar layouts. While this may not be ideal for a portfolio website, the creator wanted to experiment on different styles to work on boostrap and css concepts that he was not the most comfortable with.

Layout & Navigation

At the top we maintain the morsofficial logo and this resizes depending on the available size of the container. Below it is the main feature image with the project title and technologies used. In large screens, three related projects will be shown on the right and clicking this will take you to the appropriate page. Below the main feature image is the write up. In large screens this will be two columns but in the smallest screens there is only a single column layout.

Other Features

Aside from the write up and navigation. The screenshot section will showcase the snapshots that show the best features of the project showcased. This layout changes from 2x3 to 6x1 to 3x2 depending on the screen and device used. Its position will also adjust from right to bottom depending on the screen. Furthermore an animation is shown when the mouse is hovered on top of the image, signifying that it is currently selected. Lastly, a modal showing the zoomed image with description is shown when clicked.

Concepts used

  • Absolute positioning and z-index
  • Display Flex
  • Bootstrap Grid System
  • @media queries
  • Modals and hover
